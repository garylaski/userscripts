# sc-mb-import.user.js
- Adds a button to import or open SoundCloud releases in MusicBrainz. Requires user to be logged in to soundcloud.
- Tested on Firefox with ViolentMonkey and GreaseMonkey.
- [INSTALL](https://github.com/garylaski/userscripts/raw/refs/heads/main/sc-mb-import.user.js)

# [experimental] mb-forward-release.user.js 
- Adds the ability to pass additional seeding information through the release page.
- The additional information is be passed through the "hash" of the location. e.g. `/release/add#edit_note=test`
- This information is parsed and displayed in the resulting release page, with a submission button that opens the corresponding seeding page for each. (may trigger popup blocker)
- Tested on Firefox with ViolentMonkey.
- [INSTALL](https://github.com/garylaski/userscripts/raw/refs/heads/main/mb-forward-release.user.js)
